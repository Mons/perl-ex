NAME

ex::provide - Simple replacement of Exporter.

SYNOPSIS

        use ex::provide
                ':all'      => [ METHODS_LIST ];
        use ex::provide
                ':all'      => [ METHODS_LIST ],
                ':explicit' => [ METHODS_LIST ];

DESCRIPTION

ex::provide makes exporting functions easier.

:all

The default exporting functions. Also may be imported by tag :all

:*

Functions for specified tag. There is no need to duplicate names in :all
